🤖 МЕТОД УСМАНОВОЙ: TELEGRAM-БОТ ДЛЯ ПРОГРЕВА
=============================================
1. Откройте @BotFather в Telegram, создайте бота, скопируйте токен.
2. Вставьте токен в config.json в поле BOT_TOKEN.
3. Установите зависимости: pip install -r requirements.txt
4. Запустите бота: python bot.py
5. В боте нажмите /start для теста.

📁 Структура:
config.json      - настройки времени и токена
bot.py           - ядро бота
posts/           - посты (post_1.json ... post_6.json)
posts/images/    - папка для картинок
users.db         - создаётся автоматически при запуске
